# AGENTS.md

## Назначение проекта

Этот репозиторий является частью мульти-проектного GitHub-портфолио разработчика.

Проект должен выглядеть как серьёзный production-like case study, а не как быстрая AI-generated демка.

## Главные правила

* Не спрашивай подтверждение дизайна, если дизайн уже описан в задаче.
* Используй утверждённое визуальное направление проекта.
* Не предлагай несколько вариантов дизайна без необходимости.
* Не переписывай проект с нуля, если можно исправить точечно.
* Не ломай существующий функционал.
* Не добавляй реальные секреты, токены или API-ключи.
* Не оставляй непрофессиональные placeholder-тексты.
* Не оставляй мусорные console.log.

## Локализация

Во всех пользовательских интерфейсах должна быть поддержка двух языков:

* ru
* en

Требования:

* Русский язык — основной/default.
* Английский язык — обязательный для GitHub-портфолио.
* Должен быть переключатель RU/EN.
* Все пользовательские тексты должны быть вынесены в словари переводов.
* Не хардкодить пользовательские строки в компонентах.
* Переводить кнопки, формы, ошибки, empty states, loading states, статусы, таблицы и навигацию.
* Не смешивать русский и английский в одном состоянии интерфейса.
* Если фреймворк поддерживает html lang, выставлять его корректно.

## Перед завершением задачи

Проверь:

* npm run build
* TypeScript errors
* broken imports
* responsive layout
* language switcher
* ru/en translations
* README.md
* .gitignore
* .env.example при необходимости
* отсутствие секретов
* отсутствие мусорных файлов

* описание проекта

* деплой на Vercel
